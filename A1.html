  1. [4 marks] Consider what would happen if the returns were predictable, lets say following an MA(1) model \(R_t = \mu + V_t + .5 V_{t-1}\), where \(V_t \sim ^{iid} N(0,\sigma_v^2)\). First, estimate the values of \(\mu\) and \(\sigma_v^2\) based on the mean and variance of your stocks daily returns, by setting \(\mu = \bar{R}\) and \(\sigma_v^2 = S_R^2 / 1.25\), where \(\bar{R},S_R^2\) are the sample mean and variance of the returns. (If you are wondering about the division by 1.25, remember that for the MA(1) model \(X_t = W_t + \theta W_{t-1}\) we have \(V[X_t]=(1+\theta^2) \sigma^2_w\).)

Suppose the returns followed an MA(1) model \(R_t = \mu + V_t + .5 V_{t-1}\), where \(V_t \sim ^{iid} N(0,\sigma_v^2)\), this is what would happen

mu = mean(dailyReturn(K.TO, type = "log"))
sigma = var(dailyReturn(K.TO, type = "log"))/1.25
n=2500
MA = stats::filter( rnorm(n+1, 0, sigma), c(1, .5), side = 1, method = "convolution") + mu

A simple trading stratagy then would show what would happen if the stock was predictable perfectly

i = 2
total = 0
total_2 = 0
while (i <= 2500) {
  if (MA[i] > 0) {
    total = total + MA[i+1]
    #print('add')

  } else{
    total = total - MA[i+1]
    #print('sub')
  }
  total_2 = total_2 + MA[i]
  i = i + 1
}
